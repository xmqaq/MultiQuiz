导入说明

Excel 和 JSON 都可以导入。首次使用建议先下载模板，再按示例填写后导入。

  • Excel：第一行为表头，列顺序固定为题目、选项 A 到选项 D、正确答案
  • JSON：支持题目数组，或 { "questions": [...] } 结构
  • 每道题都需包含 question、optionA 到 optionD、answer
  • answer 仅支持 A / B / C / D，本平台导出的 JSON 也可直接重新导入
