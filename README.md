# CISP-PTE 练习题库系统

一个功能完整的本地化练习题库系统，专为CISP-PTE认证考试设计，支持Excel题库导入、模拟考试、错题管理等功能。

## 🌟 主要功能

### 📁 题库导入
- 支持Excel文件(.xlsx, .xls)拖拽上传
- 自动识别题目、选项和答案
- 实时数据验证和错误提示
- 本地存储，无需重复导入

### 📚 题库浏览
- 完整题库展示
- 关键词搜索功能
- 清晰的题目和答案显示
- 响应式设计，支持移动端

### ✏️ 模拟考试
- 自定义题目数量(20/50/100/全部)
- 可设置考试时间(30-120分钟)
- 实时倒计时提醒
- 自动评分和成绩分析
- 键盘快捷键支持

### ❌ 错题本
- 自动收集错题
- 显示错误答案和正确答案对比
- 错题时间记录
- 一键清空功能

### 📊 学习统计
- 总题数统计
- 练习进度跟踪
- 正确率分析
- 错题数量统计

## 📋 Excel格式要求

请确保您的Excel文件按以下格式组织：

| 列号 | 内容 | 示例 |
|------|------|------|
| A列 | 题目内容 | 以下哪项不是网络安全的基本要素？ |
| B列 | 选项A | 机密性 |
| C列 | 选项B | 完整性 |
| D列 | 选项C | 可用性 |
| E列 | 选项D | 便利性 |
| F列 | 正确答案 | D |

**注意事项：**
- 第一行可以是表头，系统会自动跳过
- 答案列必须填写A、B、C、D中的一个
- 确保每道题的所有信息都完整填写

## 🚀 使用方法

### 1. 启动应用
直接双击 `index.html` 文件，在浏览器中打开即可使用。

### 2. 导入题库
1. 点击「导入题库」标签
2. 拖拽Excel文件到上传区域，或点击选择文件
3. 等待系统解析完成
4. 系统会自动跳转到题库浏览页面

### 3. 开始练习
1. 在「总题库」中浏览所有题目
2. 使用搜索功能快速定位特定题目
3. 查看题目内容和正确答案

### 4. 模拟考试
1. 点击「模拟答题」标签
2. 设置题目数量和考试时间
3. 点击「开始考试」
4. 使用鼠标点击或键盘快捷键答题
5. 考试结束后查看成绩和错题分析

### 5. 错题复习
1. 点击「错题本」标签
2. 查看所有答错的题目
3. 对比错误答案和正确答案
4. 重点复习薄弱知识点

## ⌨️ 键盘快捷键

在考试模式下支持以下快捷键：

- `←` / `→` : 上一题/下一题
- `1` / `A` : 选择选项A
- `2` / `B` : 选择选项B
- `3` / `C` : 选择选项C
- `4` / `D` : 选择选项D

## 💾 数据存储

所有数据都存储在浏览器本地，包括：
- 导入的题库数据
- 错题记录
- 练习统计

**注意：** 清除浏览器数据会导致所有记录丢失，建议定期备份Excel源文件。

## 🆘 常见问题

**Q: 导入Excel文件失败怎么办？**
A: 请检查文件格式是否正确，确保答案列只包含A、B、C、D，且每道题信息完整。

**Q: 数据会丢失吗？**
A: 数据存储在浏览器本地，除非主动清除浏览器数据，否则不会丢失。

**Q: 可以导入多个Excel文件吗？**
A: 每次导入会覆盖之前的数据，建议将所有题目整理到一个Excel文件中。

**Q: 支持哪些浏览器？**
A: 支持所有现代浏览器，推荐使用Chrome或Firefox以获得最佳体验。
