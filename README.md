# CISP-PTE 练习题库系统

一个功能完整的本地化练习题库系统，专为CISP-PTE认证考试设计。

## 主要功能

### 题库管理
- **多格式导入**：支持Excel文件(.xlsx, .xls)和JSON配置文件导入
- **数据验证**：自动验证题目格式，确保数据完整性
- **本地存储**：所有数据存储在浏览器本地，无需重复导入
- **题库导出**：支持导出为JSON配置文件和CSV格式

### 题库浏览
- **完整展示**：浏览所有导入的题目和答案
- **关键词搜索**：快速定位特定题目内容
- **题库统计**：显示题库信息和数据概览

### 模拟考试
- **灵活设置**：自定义题目数量(20/50/100/全部题目)
- **时间控制**：可设置考试时间(30-120分钟)，支持倒计时
- **随机出题**：题目随机排序，模拟真实考试环境
- **实时答题**：支持鼠标点击和键盘快捷键答题
- **成绩分析**：考试结束后显示详细成绩和错题统计

### 错题管理
- **自动收集**：考试中的错题自动添加到错题本
- **错题练习**：专门的错题练习模式，针对性复习
- **智能管理**：错题练习中答对的题目自动从错题本移除
- **详细对比**：显示用户答案与正确答案的对比
- **时间记录**：记录每道错题的产生时间

### 学习统计
- **练习数据**：总题数、已练习题数、正确率统计
- **错题追踪**：错题数量变化趋势
- **学习进度**：可视化学习进度和成果

## 📋 Excel格式要求

请确保您的Excel文件按以下格式组织：

| 列号 | 内容 | 示例 |
|------|------|------|
| A列 | 题目内容 | 以下哪项不是网络安全的基本要素？ |
| B列 | 选项A | 机密性 |
| C列 | 选项B | 完整性 |
| D列 | 选项C | 可用性 |
| E列 | 选项D | 便利性 |
| F列 | 正确答案 | D |

**注意事项：**
- 第一行可以是表头，系统会自动跳过
- 答案列必须填写A、B、C、D中的一个
- 确保每道题的所有信息都完整填写

## 使用方法

### 快速开始
1. 双击 `index.html` 文件在浏览器中打开
2. 在「导入题库」页面上传Excel文件或JSON配置文件
3. 系统自动解析并验证数据格式
4. 开始使用各项功能

### 核心功能使用

**题库导入**
- 拖拽Excel文件到上传区域或点击选择文件
- 支持追加导入和覆盖导入两种模式
- 自动跳过格式错误的题目并提示

**模拟考试**
- 设置题目数量和考试时间后开始考试
- 支持键盘快捷键快速答题
- 考试结束自动评分并收集错题

**错题练习**
- 在错题本页面点击「错题练习」开始专项练习
- 答对的错题会自动从错题本中移除
- 可反复练习直到完全掌握

## 键盘快捷键

考试模式下的快捷键：
- `←` / `→` : 上一题/下一题
- `1` / `A` : 选择选项A
- `2` / `B` : 选择选项B  
- `3` / `C` : 选择选项C
- `4` / `D` : 选择选项D

## 技术特性

- **本地运行**：纯前端实现，无需服务器
- **数据存储**：所有数据存储在浏览器本地
- **跨平台**：支持Windows、macOS、Linux
- **浏览器兼容**：支持Chrome、Firefox、Safari、Edge
- **响应式设计**：适配桌面和移动设备
- **离线使用**：无需网络连接

## 常见问题

**Q: 导入Excel文件失败怎么办？**
A: 检查文件格式，确保答案列只包含A、B、C、D，且每道题信息完整。

**Q: 数据会丢失吗？**
A: 数据存储在浏览器本地，清除浏览器数据会导致丢失，建议定期备份。

**Q: 可以导入多个Excel文件吗？**
A: 支持追加导入和覆盖导入两种模式，可根据需要选择。
